# Feza_Elias_USSD-SMS

=== Here's a short description===
This is the USSD system designed for user registration, balance tracking, and transactions, with SMS notifications via Africa’s Talking. After registration, each user starts with an initial balance of 400.00 RWF. Agents register themselves via a web interface and assist users with deposits and withdrawals. All data is stored securely in a MySQL database.

Key Components:====

*USSD User Registration: Users register through a USSD menu and receive an initial balance of 400.00 RWF.

*Agent-Assisted Transactions: Users cannot deposit or withdraw money on their own. Agents facilitate all deposit and withdrawal operations via USSD.

*SMS Notifications: Every transaction triggers an SMS confirmation to the user.

*Web Portal for Agents: Agents use a web interface for their personal registration.

*Data Management: All users, agents, and transactions are stored in a MySQL database.

Roles:===

*User: Registers via USSD, starts with a balance of 400.00 RWF, and relies on agents for deposits and withdrawals.

*Agent: Registers via a web portal so that to be able to help users perform their cash withdrawal or money depoit via USSD.

Technologies Used:===

*PHP (Core application logic)


*MySQL (Data storage)

*Africa’s Talking (SMS gateway for notifications)

*HTML/CSS (Agent web interface)
